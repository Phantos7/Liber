Witaj w panelu edycji

Żeby zapisywać zmiany na stronie, potrzebujesz tokenu z GitHuba. Tworzysz go raz, przeglądarka go zapamięta.

Jak wygenerować token? (3 minuty)
  1. Wejdź na github.com/settings/personal-access-tokens/new
  2. Token name: wpisz „LIBER edycja"
  3. Expiration: wybierz „No expiration" albo zaznacz datę w przyszłości
  4. Repository access: wybierz „Only select repositories" → wskaż Phantos7/Liber
  5. Repository permissions: rozwiń „Contents" → ustaw na „Read and write"
  6. Na dole kliknij Generate token
  7. Skopiuj wygenerowany ciąg i wklej tutaj. Zapamiętaj go też w bezpiecznym miejscu — drugi raz nie zobaczysz.

Token zostaje tylko w pamięci Twojej przeglądarki. Nie wysyłamy go nigdzie poza serwery GitHuba.
