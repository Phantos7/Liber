Setup logowania

JEDNORAZOWA KONFIGURACJA

Tu generujesz zaszyfrowany plik z tokenem GitHub. Joanna po założeniu już go nie potrzebuje – będzie się logować hasłem, które tutaj ustalisz. Token zaszyfrowany jest tak, że nawet znając zaszyfrowany ciąg nie da się go odczytać bez hasła.

Krok 1 — Wygeneruj swój token GitHub (5 minut)

  1. Wejdź na github.com/settings/personal-access-tokens/new
  2. Token name: LIBER edycja
  3. Expiration: wybierz datę za rok (max dla fine-grained)
  4. Repository access: Only select repositories → wybierz Phantos7/Liber
  5. Repository permissions → Contents → Read and write
  6. Kliknij Generate token i skopiuj wartość (zaczyna się od github_pat_)

Krok 2 — Wklej token i ustaw hasło

Polecam min. 12 znaków, mieszanka liter, cyfr i znaków. Zapisz w bezpiecznym miejscu — nie da się go odzyskać.
